[[repeat1]]
== repeat&#40;String, Number&#41;: String

Repeats a `text` the amount of specified `times`


_Introduced in DataWeave 2.2.0. Supported by Mule 4.2 and later._

=== Parameters

 [%header, cols="1,3"]
 |===
 | Name | Description
 | `text` | The input string.
 | `times` | number of times to repeat char, negative treated as zero
 |===

=== Example

This example shows how the `repeat` behaves under different inputs and sizes.

==== Source

[source,DataWeave, linenums]
----
import repeat from dw::core::Strings
---
{
  "a": repeat("e", 0),
  "b": repeat("e", 3),
  "c": repeat("e", -2)
}
----
==== Output
[source,JSON,linenums]
----
{
  "a": "",
  "b": "eee",
  "c": ""
}
----
